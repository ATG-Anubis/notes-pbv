>[!Question]- C'est quoi une organisation sociale patrilinéaire ?
>- C'est une organisation sociale dans laquelle les enfants adoptent le nom de famille de leur père.

>[!Question]- C'est quoi une organisation sociale patriarcale ?
>- C'est une organisation sociale dans laquelle les hommes sont les plus importants.

>[!Question]- C'est quoi une organisation sociale matrilinéaire ?
>- C'est une organisation sociale dans laquelle les enfants adoptent le nom de famille de leur mère.

>[!Question]- C'est quoi une organisation sociale matriarcale ?
>- C'est une organisation sociale dans laquelle les femmes sont les plus importantes.

### Iroquoiens

>[!Question]- Qui a les tâches les plus importantes chez les Iroquoiens ?
>- Les femmes

>[!Question]- Quelle est la tâche la plus importante pour la survie du groupe chez les Iroquoiens ?
>- L'agriculture

>[!Question]- Qui fait la chasse, fait les échanges et défendent la famille chez les Iroquoiens ?
>- Les hommes

>[!Question]- Quelle est la structure sociale des Iroquoiens ?
>- Matrilinéaire

>[!Question]- Qui sont les chefs civils ?
>- Ce sont des hommes qui prennent les grandes décisions politiques chez les Iroquoiens.

>[!Question]- Qui choisissent les chefs civils ? (3 réponses)
>- Les femmes
>- Les anciens
>- Les chefs de guerre


### Algonquiens

>[!Question]- C'est quoi une bande ?
>- C'est un groupe de familles qui est trop petit pour être un village.

>[!Question]- Comment appelle-t-on tous les villages d'une même famille iroquoienne ?
>- Une nation

>[!Question]- Qui a les tâches les plus importantes chez les Algonquiens ?
>- Les hommes

>[!Question]- Quelle est la tâche la plus importante pour la survie du groupe chez les Algonquiens ?
>- La chasse et la pêche.

>[!Question]- Qui a le rôle de cueillie les fruits, préparer la nourriture et préparer les déplacements ?
>- Les femmes

>[!Question]- Quelle est la structure sociale des Algonquiens ?
>- Patrilinéaire

>[!Question]- Est-ce que les Algonquiens peuvent vivre en grand groupe toute l'année ?
>- Non, ils sont en grandes bandes d'été en été et ils sont en petites bandes sans chef en hiver.

>[!Question]- Quel est le nom du grand chef d'été ?
>- Le Sagamo

>[!Question]- Qui prend les décisions chez les Algonquiens ?
>- Le consensus



>[!Question]- C'est quoi le don et contre-don ?
>- C'est une tradition dans laquelle les chefs et les familles s'offrent des cadeaux lors de leur rencontre en signe de respect.

>[!Question]- Qui est responsable pour éduquer les jeunes ?
>- Tout le monde

>[!Question]- Quelle religion possède les autochtones ?
>- L'animisme, la croyance que tout possède un esprit

>[!Question]- Quelle est la fonction d'un capteur de rêves ?
>- C'est un objet qui capture les mauvais rêves pour laisser passer les bons rêves.
