# Les changements physiques

- C'est quoi un changement physique?
    - C'est une transformation de la matière mais qui ne change pas sa nature
- Quels sont les types de changements physiques?
    - Changements de phase ou d'état
    - Mélanges
- Comment se nomme le changement quand un solide devient liquide?
    - La fusion
- Comment se nomme le changement quand un liquide devient solide?
    - La solidification
- Comment se nomme le changement quand un liquide devient un gaz?
    - La vaporisation
    - L'évaporation
    - L'ébullition
- Comment se nomme le changement quand un gaz devient un liquide?
    - La condensation liquide
- Comment se nomme le changement quand un gaz devient solide?
    - La condensation solide
- Comment se nomme le changement quand un solide devient un gaz?
    - La sublimation
