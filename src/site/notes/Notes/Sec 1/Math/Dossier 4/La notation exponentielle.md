# La notation exponentielle

- C’est quoi la notation exponentielle?
    
    C’est la forme d’écriture qui utilise l’exposant
    
- Dans “5^2”, comment appelle-t-on le “5”?
    
    La base
    
- Dans “5^2”, comment appelle-t-on le “2”?
    
    l’exposant
    
- Dans “5^2=25”, comment appelle-t-on le “25”?
    
    la puissance
    
- C’est quoi un nombre au carré?
    
    C’est un nombre exposant 2
    
- C’est quoi un nombre au cube?
    
    C’est un nombre exposant 3
    
- Ça fait quoi un nombre exposant 1?
    
    La même base
    
- Ça fait quoi un nombre exposant 0?
    
    1
    
- Ça fait quoi un nombre dont la base est 1?
    
    1
