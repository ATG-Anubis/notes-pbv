# Addition, soustraction, multiplication et division des nombres décimaux

- Qu'est-ce qui est important dans la manière de présenter une addition/soustraction de nombre décimal?
    - On aligne les nombres

- Qu'est-ce qu'on fait si il y a un espace à la fin des nombres décimaux?
    - On le remplace par un 0

- Comment se nomme le résultat de l'addition?
    - La somme

- Comment se nomme le résultat de la soustraction?
    - La différence

- Comment se nomme le 1er nombre dans une multiplication?
    - La multiplicande

- Comment se nomme le 2ème nombre dans une multiplication?
    - Le multiplicateur

- Comment se nomme le dernier nombre dans une multiplication?
    - Le produit

- Quelle sont les étapes quand on multiplie un nombre décimal?
    1. Multiplier le nombre sans la virgule
    2. Remettre la virgule au bon endroit

- Comment on fait pour savoir où remettre la virgule?
    - On additionne le nombre de nombres dans la partie décimale du calcul, et on mets la virgule a la droite

- Qu'est-ce qu'on fait quand on multiplie un nombre par 10?
    - On déplace la virgule vers la droite le nombre de 0 dans le multiplicateur

- Comment se nomme le résultat de la division?
    - Le quotient

- Comment se nomme le 1er nombre dans la division?
    - La dividende

- Comment se nomme le 2ème nombre dans la division?
    - Le diviseur
