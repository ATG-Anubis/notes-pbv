# Les préfixes pour les grands nombres

- C'est quoi le préfixe pour dix?
    - Déca (da)
- C'est quoi le préfixe pour cent
    - Hecto (h)
- C'est quoi le préfixe pour mille
    - Kilo (k)
- C'est quoi le préfixe pour million
    - Méga (M)
- C'est quoi le préfixe pour milliard
    - Giga (G)
- C'est quoi le préfixe pour billion
    - Téra (T)
- C'est quoi le préfixe pour billiard
    - Péta (P)
- C'est quoi le préfixe pour trillion
    - Exa (E)
- C'est quoi le préfixe pour trilliard
    - Zetta (Z)
- C'est quoi le préfixe pour quadrillion
    - Yotta (Y)
