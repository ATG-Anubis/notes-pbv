# Le tant pour cent (pourcentage d'un nombre)

- C'est quoi le pourcentage?
    - C'est une comparaison par rapport à 100
- Qu'est-ce que ça veut dire le mot "de" entre deux nombres?
    - Ça représente une multiplication
- Quelles sont les méthodes possibles pour travailler le pourcentage?
    - Le nombre décimal
    - La fraction
    - Le produit croisé
