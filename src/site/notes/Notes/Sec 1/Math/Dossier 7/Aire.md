# Aire

- C’est quoi l’aire?
    
    C’est la mesure de la surface (superficie)
    
- Quel est le symbole pour l’aire?
    
    A
    
- Quelle est la formule pour le carré?
    
    A = c*c
    
- Quelle est la formule pour le rectangle?
    
    A = b*h
    
- Quelle est la formule pour le parallélogramme?
    
    A = b*h
    
- Quelle est la formule pour le triangle?
    
    A = b*h/2
    
- Quelle est la formule pour le losange?
    
    A = D*d/2
    
- Quelle est la formule pour le trapèze?
    
    A = (B+b)*h/2
